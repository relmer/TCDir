# Quickstart: PowerShell Alias Configuration

**Feature**: 005-powershell-aliases | **Date**: 2026-03-25

## Module Overview

| Module | File(s) | Responsibility |
|--------|---------|---------------|
| **ProfilePathResolver** | `ProfilePathResolver.h/.cpp` | Detect parent PS version, resolve 4 profile paths for that version |
| **ProfileFileManager** | `ProfileFileManager.h/.cpp` | Read/write/backup profile files, parse/find/replace marker blocks |
| **AliasBlockGenerator** | `AliasBlockGenerator.h/.cpp` | Generate PowerShell function code from SAliasConfig |
| **TuiWidgets** | `TuiWidgets.h/.cpp` | Text input, radio buttons, checkboxes, confirmation — raw console input |
| **AliasManager** | `AliasManager.h/.cpp` | Orchestrate the three flows: set, get, remove |

## Key Integration Points

### CommandLine: Adding new switches

In `CommandLine.h`, add bool members:
```cpp
bool m_fSetAliases    = false;
bool m_fGetAliases    = false;
bool m_fRemoveAliases = false;
bool m_fWhatIf        = false;
```

In `CommandLine.cpp`, add to `s_krgLongSwitches[]`:
```cpp
{ L"set-aliases",    &CCommandLine::m_fSetAliases    },
{ L"get-aliases",    &CCommandLine::m_fGetAliases    },
{ L"remove-aliases", &CCommandLine::m_fRemoveAliases },
{ L"whatif",         &CCommandLine::m_fWhatIf        },
```

In `ValidateSwitchCombinations()`, enforce:
- `--set-aliases`, `--get-aliases`, `--remove-aliases` mutually exclusive with each other and with listing switches
- `--whatif` only valid alongside `--set-aliases` or `--remove-aliases`

### TCDir.cpp: Dispatch

Early in `ProcessCommandLine()`, after switch parsing, before directory listing:
```cpp
if (cl.m_fSetAliases || cl.m_fGetAliases || cl.m_fRemoveAliases)
{
    CHR(CAliasManager::Run(cl));
    goto Error;  // Clean exit, skip directory listing
}
```

### Generated Alias Block (example output)

```powershell
# >>> tcdir aliases — DO NOT EDIT: this block is managed by tcdir and will be replaced <<<
# Generated by tcdir v1.8.0
function d { tcdir @args }
function dd { d /a:d @args }
function ds { d /s @args }
function dsb { d /s /b @args }
function dw { d /w @args }
# >>> end tcdir aliases <<<
```
